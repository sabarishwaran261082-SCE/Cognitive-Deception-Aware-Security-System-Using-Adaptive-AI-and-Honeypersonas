# CDASS Project Overview

CDASS (Cognitive Deception Aware Security System) is an AI-driven cybersecurity framework deployed on AWS.

The system uses adaptive honeypersonas, machine learning, behavioral analysis, and deception techniques to identify suspicious activities and protect production services.

Main Components:

- Application Load Balancer
- Detection Engine
- Adaptive Honeypersona Environment
- Containerized AI Runtime
- SageMaker ML Training
- Amazon S3
- Amazon DynamoDB
- Amazon SQS
- Amazon SNS

The architecture is deployed entirely inside a secure Amazon VPC using private subnets and VPC endpoints.
